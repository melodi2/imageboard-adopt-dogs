# imageboard-adopt-dogs

This "Adopt a dog" Imageboard is a Vue.js-based single page application where users can upload, comment and delete pictures of dogs who are looking for a new home.

## Features

-   Upload images
-   Display images in a modal and add comments
-   Delete image and the associated comments

## Technology Stack

-   Vue.js
-   Node.js, Express
-   PostgresSQL
-   AWS
-   JavaScript
